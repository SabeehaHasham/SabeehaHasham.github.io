## Results
This project aims to focus on the design and implementation of a robot that is required to pick up one of five objects and place it in its respective container. Vision detection has been implemented using a GoogLeNet model aiming to include shape detection, when issues were faced alternatives were explored such as colour detection using Opencv. Training a model will be revisited however.
